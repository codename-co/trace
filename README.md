# Tracé

Record API traffic and export it as a ready-to-use OpenCollection. One click on, one click off.

## Install

1. `npm install`
2. Go to `chrome://extensions` → Enable **Developer mode** → **Load unpacked** → select the `public/` folder

## Usage

1. Click the extension icon to **start** recording
2. Browse the site
3. Click the icon again to **stop** — the recorded collection opens as interactive API documentation
4. Right-click the icon → **Options** to enable **Smart mode** (auto-links response values across requests)
